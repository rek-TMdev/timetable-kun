# インストーラー定義

このフォルダには公開用に相対パス化したビルド定義を置いています。

- `時間割くん.spec`: 本体アプリ用 PyInstaller 定義
- `時間割くんツール.spec`: ドライバとチェッカー用 PyInstaller 定義
- `時間割くん.iss`: 本体アプリ用 Inno Setup 定義
- `時間割くんツール.iss`: ドライバとチェッカー用 Inno Setup 定義

実運用版の絶対パス、生成済み `exe`、`_internal`、実データは公開対象外です。配布前にはバージョン番号、発行者名、出力先、コード署名、更新方法を環境に合わせて調整してください。
