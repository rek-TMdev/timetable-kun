時間割くん HTMLマニュアル概要

このページは GitHub 公開用の概要版です。実運用版の生成済み HTML マニュアルには学校固有の説明や連絡先が含まれるため、このリポジトリには同梱していません。

公開方針: 実在校名、実データ、個人情報、運用先固有の設定は含めません。

本体マニュアル

  • 履修条件の選択
  • 時間割候補の生成
  • 手動固定科目の指定
  • 競合・重複・前提科目の確認
  • Excel への出力

ドライバマニュアル

  • 学校別設定ファイルの作成
  • 教科マスタ、別名、前提科目、同時履修不可条件の編集
  • 3-way マージによる設定競合の解決
  • 配布用設定の更新

チェッカーマニュアル

  • 設定ファイルの整合性確認
  • 保存済み時間割の確認補助
  • 移行・配布前のチェック

インストールと更新

  • installer/ の PyInstaller 定義で実行ファイルを作成
  • Inno Setup 定義で Windows インストーラーを作成
  • 実運用環境では配布先、更新手順、署名を別途設定
